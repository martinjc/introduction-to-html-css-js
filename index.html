About

These notes follow on from the Introduction to HTML and CSS and the Introduction to JavaScript and start to put the three technologies together so we can start to build rich interactive front-end websites. We do not look at any specific ‘flavour of the moment’ front-end development libraries, those looking for information on React, Vue, Angular or whatever else is currently ‘hip’ in the ‘overuse of JavaScript is web development’ world should look elsewhere. Here we keep our focus on clean, semantic HTML and CSS with the sparing addition of vanilla JavaScript where absolutely necessary to provide functionality.
